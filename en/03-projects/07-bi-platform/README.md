# Self-hosted BI Platform

[Deutsch](../../../03-projects/07-bi-platform/README.md) · **English**

## Project

**Self-hosted Metabase** for a large classifieds marketplace — dashboards, Prometheus (5-min metrics), weekly backups, Certbot TLS.

| | |
|---|---|
| **Period** | 2025 |
| **Role** | Freelance — infrastructure, BI, operations |

## Lessons Learned

- Self-hosted Metabase works for mid-sized businesses when monitoring and backups are built in from day one
- Dashboard work + infrastructure in one engagement = faster business value
